# DPT Surabaya Endpoint

Berisikan mengenai informasi Daftar Pemilih Tetap Kota Surabaya


## How to setup

1. `$ bundle install`
2. Configure your database credentials in database.yml
2. `$ bundle exec rake db:setup`
3. `$ bundle exec rails server`
